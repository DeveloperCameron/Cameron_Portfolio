# Cameron Henry — Developer Portfolio

A personal portfolio site built from scratch with a warm editorial aesthetic. Designed to showcase my projects, skills, and background as a front-end developer.

🔗 **Live Site:** [developercameron.github.io/Cameron_Portfolio](https://developercameron.github.io/Cameron_Portfolio/)

---

## About

I'm Cameron, a 19-year-old front-end developer based in Texas. I'm currently studying Web Design & Development at TSTC while independently building projects and completing industry certifications. This portfolio is the hub that ties everything together.

---

## Built With

- HTML5
- CSS3 (Flexbox, Grid, custom animations)
- Vanilla JavaScript
- Google Fonts — Playfair Display, DM Sans
- Deployed on GitHub Pages

## Features

- Warm editorial design — magazine-style layout with serif typography
- Scroll-reveal animations triggered by IntersectionObserver
- Fully responsive — mobile first
- Smooth scroll navigation with sticky header
- Projects section linking to all live deployed work
